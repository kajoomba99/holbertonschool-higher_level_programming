Usando 'my_list'
------------------

First lets import our class

    >>> MyList = __import__('1-my_list').MyList

Normal cases:

	>>> my_list = MyList()
	>>> my_list.print_sorted()
	[]
	>>> my_list.append(1)
	>>> my_list.append(4)
	>>> my_list.append(2)
	>>> my_list.append(3)
	>>> my_list.append(5)
	>>> my_list.print_sorted()
	[1, 2, 3, 4, 5]
	>>> my_list.append(0)
	>>> my_list.print_sorted()
	[0, 1, 2, 3, 4, 5]
	>>> my_list.append(-3)
	>>> my_list.print_sorted()
	[-3, 0, 1, 2, 3, 4, 5]
	>>> my_list = MyList({1, 2, 3, 4})
	>>> my_list.print_sorted()
	[1, 2, 3, 4]
	>>> my_list = MyList((1, 2, 3, 4))
	>>> my_list.print_sorted()
	[1, 2, 3, 4]

Error cases:

	>>> my_list = MyList()
	>>> my_list.append("x")
	>>> my_list.append("y")
	>>> my_list.append(1)
	>>> my_list.print_sorted()
	Traceback (most recent call last):
	TypeError: '<' not supported between instances of 'int' and 'str'
	>>> my_list = MyList()
	>>> my_list.append()
	Traceback (most recent call last):
	TypeError: append() takes exactly one argument (0 given)
